# Jury's Bar Management System

A Point of Sale (POS) system built with C# and WPF using SQLite for database.

## Features
- User authentication & role-based access
- Product management
- Sales with receipts
- Payments and payroll
- User account management

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Build and run
4. Database is auto-generated

## Default Admin
- Username: admin
- Password: admin

## Database Structure
- Users
- Products
- Sales
- Payments
- Payroll
